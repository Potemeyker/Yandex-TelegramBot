KinoBot - это Telegram-бот, который помогает пользователям находить, скачивать и смотреть фильмы и сериалы.

Структура:
- data/: Содержит json файлы фильмов, связанные с видео контентом.
- database/: Включает в себя файл базы данных SQLite (`telebot.db`), содержащий информацию о фильмах (films) и пользователях (users).
- bot.py: Основной файл, содержащий функциональность бота.
- database.py: Файл, обрабатывающий операции с базой данных.

Бота можно найти по нику в тг @YandexKinoBot или перейти по ссылке https://t.me/@YandexKinoBot

После того как пользователь нажимает на кнопку старт, его id добавляется в БД в таблицу users (для осуществления доп функционала).

Чтобы найти интересующийся фильм или сериал, нужно отправить боту названия фильма сейчас для поиска доступно фильм Довод
и сериал Fallout. Такой маленький список обусловлен ограниченностью Telegram: видео, отправляемое ботом не может
превышать 50 мб. Поэтому приходиться с личного клиента сначала скачать фильм во всех качествах и озвучках (а у сериалов ещё все серии),
а потом отправить на сервера Telegram любым сообщением, скопировать его id и сохранять для каждой серии, озвучки и качества.

После того как пользователь вводит название фильма в любом регистре на любом языке, делается запрос в БД в таблице films по tags (название на
английском и русском языке) и возвращает путь к json, где хранятся id видео контента. Функционал базы данных осуществлён в файле database.py

Из json файла берётся id нужной видео и отправляется пользователю, так же появляются кнопки выбора озвучки, качества, поиска, и для сериалов
можно выбрать серию. В случае если id нет, то пользователю вместо видео выводиться постер фильма для защиты от ошибок.

Пользователь может посмотреть отправленное видео, и так ка Telegram хранить файлы на устройстве оно сразу скачиваться. Но пользователь
так же может сохранить видео себе в галерею.

В будущем планируется добавить больше фильмов, историю поиска, избранное, рекомендации по просмотренным фильмам и добавление vip подписки или рекламы.
